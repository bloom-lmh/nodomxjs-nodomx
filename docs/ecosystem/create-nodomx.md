# `create-nodomx`

官方脚手架包名：`create-nodomx`

## 使用

```bash
npm create nodomx@latest my-app
```

## 默认得到什么

- `.nd` 入口项目
- Rollup 配置
- 官方 dev server
- 基础样式与示例页面

## 常用选项

脚手架支持这些常见参数：

- `--install`
- `--package-mode registry`
- `--package-mode local`
- `--force`

## 什么时候用它

如果你想快速开始一个 NodomX 项目，脚手架永远是最省心的入口。

如果你已经有现成的 Vite / Rollup 工程，再单独装插件会更灵活。
